# No Gap Chrome Extension
Removes so called `gender gaps` from web pages (Russian language). Makes text readable. 

In example, before:
```
Конференция пропагандисто_к
Говорили с журналист_ками, иностранными агент_ками и активист_ками об их работе...
```

After:
```
Конференция пропагандисток
Говорили с журналистками, иностранными агентками и активистками об их работе...
```
